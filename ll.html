Fully offline Android launcher — Soul Browser ya kisi bhi browser mein HTML file open karo. Koi server nahi, koi internet dependency nahi.
⭕ Circles
🕐 Clock — Live time + battery + network dot
📊 Dashboard — Usage stats, storage, backup status
🔗 My Links — Saved URLs, categories
🔎 Search — Web search + custom shortcuts
📦 Storage — Files browser ke andar store karo
🌐 Redirect — Auto ya manual URL redirect on launch
➕ Add App — HTML file = circle ban jaata hai
📱 Custom Apps
HTML upload karo → circle banta hai → tap = fullscreen.
Long press → rename, delete, replace, hide
Duplicate check — same file dobara nahi chadhega
🔎 Search Commands
/set → Settings   /my → My Links
/dash → Dashboard   /box → Storage
yt cats → YouTube search (custom shortcut)
